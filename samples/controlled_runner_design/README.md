# Controlled Runner Design Samples

This directory contains design-only artifacts for
`common-foundation-controlled-runner-design-v1`.

- `controlled_runner_design_v1.json`: capability ladder, authority boundaries,
  command class taxonomy, future unlock gates, and evidence contract.
- `controlled_runner_readiness_review_v1.json`: compact review readback showing
  why the future probe is not approved by this slice.

These samples are not executable manifests. They must not be interpreted as
permission to run commands, create a runner, schedule work, or write target
repositories.
